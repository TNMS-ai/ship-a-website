# Typography

## Font stack

This starter ships with **system fonts only** — zero external requests, zero
layout shift. See `src/styles/AGENTS.md` for instructions on adding custom fonts.

```css
--font-sans: ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont,
  "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif;
--font-mono: ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas,
  "Liberation Mono", "Courier New", monospace;
```

## CSS primitives

These class names are defined in `global.css` and enforced by BEM linting.
Use them — do not invent new typographic classes.

| Class | Usage |
|---|---|
| `site-headline site-headline--display` | Large display heading (below hero) |
| `site-headline site-headline--hero` | Largest hero heading |
| `site-eyebrow` | Small uppercase label, dark background |
| `site-eyebrow--paper` | Small uppercase label, light background |
| `site-kicker` | Mono numbered section label, dark background |
| `site-kicker--paper` | Mono numbered section label, light background |
| `site-accent--cyan` | Neon glow halo around a word (cyan) |
| `site-accent--purple` | Neon glow halo around a word (purple) |
| `site-accent--magenta` | Neon glow halo around a word (magenta) |
| `site-accent--green` | Neon glow halo around a word (green) |

## Type scale tokens

All defined in `@theme {}` in `global.css`, automatically available as
Tailwind utility classes (`text-xs`, `text-sm`, … `text-7xl`).

| Token | Size | Line height |
|---|---|---|
| `--text-xs` | 0.75rem | 1.5 |
| `--text-sm` | 0.875rem | 1.5 |
| `--text-base` | 1rem | 1.6 |
| `--text-lg` | 1.125rem | 1.5 |
| `--text-xl` | 1.25rem | 1.4 |
| `--text-2xl` | 1.5rem | 1.3 |
| `--text-3xl` | 1.875rem | 1.2 |
| `--text-4xl` | 2.25rem | 1.15 |
| `--text-5xl` | 3rem | 1.1 |
| `--text-6xl` | 3.75rem | 1.05 |
| `--text-7xl` | 4.5rem | 1 |
